# PowerPoint Templates

Place your `.pptx` template files in this folder to use them with the TEXT to PPT Generator.

## Template Guidelines:

### Required Slide Layouts:
1. **Layout 0**: Title Slide (with title and subtitle placeholders)
2. **Layout 1**: Content Slide (with title and content/bullets placeholder)
3. **Layout 2**: Section Header (optional - for section dividers)
4. **Last Layout**: Thank You / Closing Slide

### Best Practices:
- Use standard PowerPoint placeholder names
- Keep layouts simple and clean
- Test your template before uploading
- Save as `.pptx` format (not `.ppt`)

## How to Add Templates:

1. Create or download a PowerPoint template
2. Save it as `.pptx` file
3. Place it in this `templates/` folder
4. It will appear in the dropdown menu automatically

## Example Templates:

You can download free templates from:
- Microsoft Office Templates
- SlidesCarnival
- SlidesGo
- Canva (export as PPTX)

Just make sure they have the standard placeholder structure!
